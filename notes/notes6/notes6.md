### Notes 6

#### Command list used in Lab 6:

* **mkdir (directory creation): It is used for creating a single directory or multiple directories.**

To create a directory it must followed the next function: 

```
In the terminal: mkdir + the name of the directory

Example: mkdir ~/cis106/notes/notes6 

(The last directory is the folder recipient)
```

* **touch (document creation): It is used for creating files.**

To create a file it must followed the next function:

```
In the terminal: touch + the name of the directory + file name.extension

Example: touch ~/cis106/notes/notes6.md

(The last part of the address is the file creation)
```
* **rm (remove files/folders): It remove files and folders.**

To create a directory it must followed the next function:

```
In the terminal: 

rm + directory + name file 
rm -r + directory + folder name

Example: rm ~/Downloads/file.txt / rm -r ~/Downloads
```

* **cp (copy files/folders): It copies files and folders.**

To create a directory it must followed the next function:

```
In the terminal: 

cp + directory + name file + final directory 
cp -r + directory + folder name + final directory

Example: cp ~/Downloads/file.txt / cp -r ~/Downloads
```

* **mv (move files/folders): It moves files and folders.**

To create a directory it must followed the next function:

```
In the terminal: 

mv + directory + name file + final directory
mv -r + directory + final directory

Example: mv ~/Downloads/file.txt ~/cis106
mv -r ~/Downloads ~/cis106
```
